# Optimization Prompt Pack

This directory is reserved for optimization-course prompts.

Current prompt files are placeholders unless populated:

- `pdf2latex.md`
- `latex_solution.md`

When these prompts are implemented, they should follow the same repository conventions:

- Keep source conversion separate from solution generation.
- Do not overwrite original source files.
- Use clear output naming rules.
- Add concise error reporting through `issues.md` when files, formulas, OCR, compilation, or dependencies fail.
